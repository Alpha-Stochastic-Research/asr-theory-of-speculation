# Project Documentation

This directory contains the internal and extended documentation for the `asr-theory-of-speculation` repository.

## Purpose

The `.docs` directory is intended to centralize documentation related to:

* the project architecture;
* mathematical and theoretical notes;
* implementation decisions;
* development and maintenance procedures;
* contribution guidelines;
* release and reproducibility processes.

## Repository

The main project documentation remains available in the root files:

* [`README.md`](../README.md)
* [`REPRODUCIBILITY.md`](../REPRODUCIBILITY.md)
* [`CITATION.cff`](../CITATION.cff)

Additional documentation may be added to this directory as the project evolves.
