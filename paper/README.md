# Working Paper

This folder contains the LaTeX source for the accompanying Alpha Stochastic Research working paper:

```text
Bachelier's Theory of Speculation Revisited:
A Reproducible Reconstruction of the Origins of Quantitative Finance
```

## Files

```text
paper/
├── main.tex
├── references.bib
└── README.md
```

## Compilation

From the `paper/` directory, compile the paper with:

```bash
pdflatex main.tex
bibtex main
pdflatex main.tex
pdflatex main.tex
```

The repeated `pdflatex` commands are needed to resolve citations, references, tables, and figure numbering.

## Required Figures

The LaTeX file expects the following figures to exist in the root `figures/` directory:

```text
figures/fig1_random_walk_martingale.png
figures/fig2_option_pricing.png
```

These figures are generated by running:

```bash
python src/brownian_motion.py
python src/option_pricing.py
```

from the root of the repository.

## Reproducibility

To reproduce the numerical results before compiling the paper:

```bash
pip install -r requirements.txt
pytest -q
python src/brownian_motion.py
python src/option_pricing.py
```

## Paper Scope

The paper provides a modern reproducible reconstruction of Louis Bachelier's 1900 *Théorie de la Spéculation*.

It includes:

- historical context;
- literature review;
- arithmetic Brownian motion model;
- martingale and variance-scaling derivations;
- Bachelier option-pricing formula;
- Monte Carlo validation;
- comparison with Black-Scholes;
- discussion of model limitations;
- open-science reproducibility framework.
